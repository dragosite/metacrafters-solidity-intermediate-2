## Backend/Smart Contract deployment:

```bash
cd backend
yarn install
npx hardhat run scripts/deploy.js
```

## Frontend:

In the parent folder:

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
